# Nouran Nasri - ATS-Friendly Digital CV

A professional, single-column, clean digital resume built with Flutter. This application provides an ATS-optimized view of Nouran Nasri's curriculum vitae, focusing on clear sections, readable fonts, and a standardized layout.

## Features
- **ATS-Optimized Layout**: Single-column design with standard, clear headings (Summary, Experience, Education, Skills).
- **Responsive Design**: Adapts beautifully to mobile, tablet, and desktop screens.
- **Clean Typography**: Uses standard, highly readable fonts and clear bullet points to ensure information is easy to parse.

## User Flows
- Launch the application to immediately view the professional resume.
- Scroll through the chronological work experience and education.
- Easily review skills and contact information at a glance.

## Tech Stack
- Flutter
- Dart
- Material Design 3

## Setup & Run
1. Ensure you have Flutter installed.
2. Clone the repository and navigate to the project root.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to launch the application.
